## Appointment with Doctor

<img width="1690" height="844" alt="image" src="https://github.com/user-attachments/assets/02506e4d-11f6-457f-8110-f6e232dfcfb4" />


A full-stack healthcare scheduling system that allows patients to book doctors, track appointments, and receive real-time updates.
Developed using Next.js, Neon Postgres, Prisma, and ShadCN UI for secure data handling and a smooth user experience..

## Tech Stack  

-Frontend: React 19, Tailwind CSS, ShadCN UI
-Backend: Next.js 15, Prisma, Arcjet
-Database: Neon Postgres
-Authentication: Clerk
-Video Calling: Vonage
